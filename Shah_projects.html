CSP Scratch Project

The goal of this project was to make a game that progressively increases in difficulty and is fun to play for a user. My partner and I decided to create a game called "The Floor is Lava", a game that infinitely increases in difficulty using various techniques in order to capture and preserve the attention of the user. My role in the project was both planning and programming. Every work day consisted of partner programming where one person would give an overall design for the structure of a program, while another would actually implement it using code. We were able to propose different ideas to solve problems we came across, and bounced them off of each other to find the most efficient solution. I learned how to use Scratch to create games, and also how to partner program effectively to create a polished final product. We had to overcome various challenges while creating the game. For example, characters were constantly going through blocks due to the faulty default collision detection. As a result, my partner and I devised a manual condition for collision detection and were able to correct mistakes while planning and implementing it. Teamwork was crucial to this project, and I was able to develop skills in working collaboratively on programming projects instead of working solo.
